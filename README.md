# Express Telegram Integrator

Сервис на Node.js для интеграции внешних данных с Telegram-ботом.

### О проекте
Демонстрация масштабируемого микросервиса с разделением ответственности.

### Инструменты
* **Node.js** / **Express**
* **Axios**
* **dotenv**

### Архитектура
* `src/config/` — конфигурация.
* `src/services/` — бизнес-логика и работа с внешними API (Telegram, JSONPlaceholder).
* `src/routes/` — эндпоинты.

### Запуск
1. Склонировать репозиторий.
2. Установить зависимости: `npm install`.
3. Создать файл `.env` и добавить свои токен бота и ID чата.
4. Запустить: `node index.js`.
